# devops-ecommerce-platform
Production-Grade E-Commerce Platform (AWS + Kubernetes)


Design, build, and deploy a high-traffic e-commerce application capable of handling 50,000+ concurrent users, with:

Full Infrastructure as Code

Automated CI/CD

Zero-downtime deployments

Autoscaling, monitoring, and disaster recovery


| Layer              | Technology                 |
| ------------------ | -------------------------- |
| Frontend           | PHP (Dockerized)           |
| Backend            | PHP REST API               |
| Cache              | Redis                      |
| Database           | MySQL (RDS Multi-AZ)       |
| Container Platform | AWS EKS                    |
| CI/CD              | Jenkins                    |
| IaC                | Terraform                  |
| Monitoring         | Prometheus + Grafana       |
| Ingress            | AWS ALB Ingress Controller |
